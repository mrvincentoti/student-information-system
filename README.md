## Framework used

- NodeJS v13.8.0
- Express v4.18.1
- Postgres
- Sequelize

### Steps to install:

#### Backend

1. clone or donwload the repository.
2. cd backend
3. Run `cp .env.example .env`
4. Update the variables in the .env file to suit you
5. Run `npm install`
6. Run `nodemon server.js`
